# Upwork Proposal Generator Skill

A Claude Code skill that generates tailored, compelling Upwork proposals designed to help freelancers stand out and win clients.

## Overview

This skill analyzes job posts and creates personalized proposals that address client needs, showcase relevant experience, and follow proven proposal structures that convert.

## Features

- **Job Post Analysis**: Extracts pain points, requirements, and hidden needs
- **Personalized Hooks**: Creates attention-grabbing openers (not generic templates)
- **Tone Matching**: Adapts proposal voice to match client's energy
- **Concise Format**: Stays under 250 words for maximum impact
- **Quality Checklist**: Ensures proposals meet best practices

## Usage

Invoke the skill with the job post content:

```
/upwork-proposal [paste the job post here]
```

## Example

```
/upwork-proposal Need a React developer to fix performance issues on our e-commerce site. Pages are loading slowly, especially the product pages. We use Next.js and have about 500 products. Budget: $500-1000.
```

## Output

The skill generates:

1. **Complete Proposal** - Ready to copy-paste
2. **Proposal Breakdown**:
   - Hook type used
   - Key differentiator
   - Tone match analysis
   - Word count

3. **Optional Variations**:
   - Shorter version (under 150 words)
   - Alternative hook angle

## Proposal Structure

| Section | Purpose |
|---------|---------|
| Opening Hook | Grab attention with insight, not "I read your post..." |
| Show Understanding | Demonstrate you've read and understood the job |
| Relevant Experience | 1-2 hyper-relevant examples with results |
| Your Approach | Brief outline of how you'd tackle the project |
| Soft CTA | Suggest next step + mention availability |

## What This Skill Avoids

- Generic templates
- Life story openers
- Overusing "I"
- Listing every skill
- Desperation signals
- Over-promising
- Walls of text

## Customization by Project Type

- **Development/Technical**: Leads with technical understanding
- **Design/Creative**: Leads with aesthetic understanding
- **Writing/Content**: Uses the proposal as a writing sample
- **Marketing/Strategy**: Leads with strategic insight
