# Social Media Optimizer Skill

A Claude Code skill that transforms a single piece of content into platform-optimized posts for Facebook, Instagram, LinkedIn, and Twitter/X.

## Overview

Write your story or announcement once, and this skill generates tailored posts for each major social media platform, complete with platform-specific formatting, tone, and hashtags.

## Features

- **Platform-Specific Optimization**: Each post follows the unique best practices for its platform
- **Hashtag Strategy**: Appropriate hashtag counts and styles for each network
- **Character Limit Compliance**: Posts stay within platform limits
- **Ready-to-Copy Output**: Quick copy section for immediate use
- **Visual Suggestions**: Recommendations for Instagram images/carousels

## Usage

Invoke the skill with your content:

```
/social-media-optimizer [your story or content]
```

## Example

```
/social-media-optimizer I just launched my new productivity app that helps developers track their coding time and identify patterns in their work habits.
```

## Output

The skill generates:

1. **Twitter/X Post** (280 chars max)
   - Punchy, attention-grabbing
   - 1-2 hashtags

2. **LinkedIn Post** (up to 3000 chars)
   - Professional, story-driven
   - 3-5 hashtags
   - Engagement question included

3. **Instagram Caption** (up to 2200 chars)
   - Authentic, emoji-friendly
   - 8-12 hashtags
   - Visual suggestion included

4. **Facebook Post**
   - Conversational, community-focused
   - 1-3 hashtags
   - Share-worthy elements

Plus:
- Hashtag master list for all platforms
- Quick copy section for each post
- Suggested posting times

## Platform Specifications

| Platform | Ideal Length | Hashtags | Tone |
|----------|--------------|----------|------|
| Twitter/X | 70-100 chars | 1-2 | Punchy, direct |
| LinkedIn | 1200-1500 chars | 3-5 | Professional, reflective |
| Instagram | 150-300 chars | 8-12 | Authentic, visual |
| Facebook | 80-250 chars | 1-3 | Conversational |

## Content Types Supported

- Achievement/Milestone announcements
- Product/Launch announcements
- Educational/How-To content
- Personal stories/Behind-the-scenes
